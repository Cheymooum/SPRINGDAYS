Pour utiliser ce site sur le serveur de l'UE (bdw1.univ-lyon1.fr), quatre étapes :

- se connecter à PHPMyAdmin, choisir l'onglet "importer", et importer le fichier "bd.sql". Cela doit créer les tables dans votre BD.

- éditer le fichier "includes/config-bd.php" et modifier les valeurs des constantes "UTILISATRICE", "MOTDEPASSE" et "BDD".

- envoyer le répertoire "Jardin" sur le serveur (soit avec scp ou rsync en ligne de commande, soit avec un outil graphique comme FileZilla).

- ouvrir un navigateur et saisir l'url "http://bdw1.univ-lyon1.fr/p1805890/Jardin/"

=========================================================

Pour utiliser ce site EN LOCAL (sur votre machine avec XAMPP ou équivalent), quatre étapes :

- se connecter à PHPMyAdmin, choisir l'onglet "importer", et importer le fichier "bd.sql". Cela doit créer les tables dans votre BD.

- éditer le fichier "includes/config-bd.php" et modifier les valeurs des constantes "UTILISATRICE", "MOTDEPASSE" et "BDD".

- déplacer le répertoire "Jardin" dans le répertoire public de XAMPP (généralement "xampp/htdocs/").

- ouvrir un navigateur et saisir l'url "http://localhost/Jardin/"
